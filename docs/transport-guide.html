Transport Components

Conveyor — $10

Moves items horizontally between machines. The basic building block of every factory.

Place between any two machines to transfer items.

Splitter — $50

Divides item flow into multiple paths. Three variants available:

L/R Splitter: Alternates items left and right (50/50 split)

Three-Way: Distributes items across three outputs (33/33/33)

Overflow: Sends items to secondary output only when primary is backed up

Filter

Routes items based on type. Essential after prospectors to separate gems from stones, or to sort different bar types.

Place after any machine that outputs mixed item types.

Elevator

Moves items between floors. Required for multi-story factory designs.

Vertical transport — connect upper and lower production floors.

Gate Belt

Controlled conveyor that can be toggled on/off. Useful for managing item flow timing and preventing backups.

Factory Layout Tips

💡 Key Principles:

🔀 Smelters Split Automatically

Ore Smelters output bars on one side and stones on another. Plan your layout to accommodate both outputs flowing to different processing lines.

🔍 Filter After Prospectors

Place filters after your prospector chain to separate gems from surviving stones. Gems go to polish → cut line; stones continue to crusher → brick/concrete line.

🏗️ Build Vertically

Use elevators to create multi-floor factories. Put ore processing on top, bar processing in the middle, and stone/byproduct lines on the bottom floor.

🔄 Use Overflow Splitters

Overflow splitters prevent backups. If a machine is full, items flow to an alternative path instead of clogging the line. Essential for high-throughput factories.

Optimal Layouts by Stage

Early Game (Levels 1–4): Single Floor Linear

Layout: Mine → Ore Cleaner → Polisher → Ore Smelter → sell bars

Stone line: Smelter stone output → Prospectors → Filter → Crusher → Brick Mold → Kiln → sell

Keep it simple. One straight line for bars, one for stones. No splitters needed yet.

Mid Game (Levels 5–6): Two-Floor Split

Top floor: Mine → Clean → Polish → Smelt → Temper → bar products (coil/plate/bolt lines)

Bottom floor: Stone line → Prospectors → Filter gems/stones → Cement Mixer → Concrete Bricks → Kiln → Polish → QA

Split bar processing into dedicated lines: one for coils (circuits, rings), one for plates (pipes, mech parts), one for bolts (frames).

Late Game (Levels 7–10): Three-Floor Multi-Line

Top floor: Ore processing → Multiple smelter lines → Tempering → Dedicated bar product lines

Middle floor: Combination machines (frames, casings, circuits, alloys) → End-game products

Bottom floor: Full stone processing (prospectors, concrete bricks, glass, ceramic) + gem processing

Use multiple output belts (2nd, 3rd, 4th) to run parallel processing. Each belt can feed a completely independent factory line.

Concrete Brick Pairing Pattern

The Pairing Problem

Concrete bricks require both dust (from crushed stone) and a whole stone in the Cement Mixer. You need to pair stones: crush one, keep one whole.

Solution: Alternating Splitter

1. Post-prospector stones → L/R Splitter

2. Left: → Crusher → dust output to Cement Mixer input A

3. Right: → Conveyor → direct stone to Cement Mixer input B

4. Cement Mixer → Brick Mold → Kiln → Polish → QA → sell

The L/R splitter ensures a 50/50 split, perfectly pairing dust and stone for cement production.

Throughput Optimization

⚡ Avoid Bottlenecks

If a machine processes slower than items arrive, place multiple copies in parallel. Use splitters before and mergers after.

📏 Belt Length Matters

Shorter conveyor runs = faster throughput. Minimize distance between machines. Don't route belts around obstacles when you can rearrange machines.

🔄 Balance Your Lines

If you split into 3 bar product lines (coil/plate/bolt), ensure each line has enough machines to handle its share of throughput.

📦 Buffer Zones

A few extra conveyors between high-throughput machines act as buffers, absorbing brief surges and preventing stalls.

🎯 Rule of thumb: When your seller is always busy, you need more sellers or higher-value products. When your machines are idle, you need more unloader capacity or better item routing.
