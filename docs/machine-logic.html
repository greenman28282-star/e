⚙️ Machine Logic & Obsolescence

Which machines stay relevant, which become obsolete, and how they interact.

💡 Key Insight: Very few machines become obsolete! Most serve different recipe chains. The main exception is the Sifter (replaced by Nano Sifter) and the Blast Furnace (a trap).

Ore Processing (Pre-Smelting)

Ore Cleaner Core

$80 — +$10 to uncleaned ores

Always clean ores before smelting. The $10 boost carries through the ×1.2 smelter multiplier.

Polisher Core

$250 — +$10 to any unpolished item

Apply to ores before smelting, and again to finished products. Universal and never obsolete.

Smelting

Ore Smelter Core

$380 — Ore → Bar (×1.2) + Stone (1 per 2 ores)

Your primary smelter forever. Produces 1 stone byproduct for every 2 ores processed. Better multiplier than Blast Furnace.

Blast Furnace Trap

$25,000 — Ore → Bar (×0.9) + Stone (1 per ore)

❌ Worse bar multiplier (×0.9 vs ×1.2). Only produces more stones — not worth the bar value loss. Avoid unless you specifically need maximum stone output.

Bar Processing

Coiler Useful

$1,750 — Bar → Coil (+$20)

Feeds circuits, rings, and electromagnets. Always needed in some production line.

Plate Stamper Useful

$3,000 — Bar → Plate (+$20)

Feeds pipes, mech parts, filigree, and casings. High demand for plates in late-game.

Bolt Machine Useful

$2,800 — Bar → Bolts (+$5)

Small flat bonus but bolts are required for frames and casings. Essential for combination chains.

Tempering Forge Core

$50,000 — Doubles bar value (×2)

Massive upgrade. Apply to bars after smelting cleaned + polished ores. Never obsolete.

Plate Products

Pipe Maker Useful

$4,000 — Plate → Pipe (+$20)

Pipes feed optics and engines. Always needed in at least one line.

Mech Parts Maker Useful

$8,000 — Plate → Mech Parts (+$30)

Best flat bonus of any plate product. Feeds into Engine Factory.

Filigree Cutter Situational

$50,000 — Plate → Filigree (+20%)

Percentage-based bonus scales with bar value. Only needed for the Glider chain.

Stone Processing

Crusher Core

$1,750 — Items → Dust ($1)

Destroys any item and produces dust worth $1. Required for all stone processing chains. Crush stones after prospecting.

Brick Mold Core

$2,500 — Dust → Brick ($25) or Cement → Concrete Brick ($100)

Dual-purpose machine. Concrete bricks are 4× more valuable than regular bricks.

Kiln Core

$4,750 — Dust → Glass ($30), Bricks ×1.2

Fires bricks (Brick $25 → Kilned $30) and converts raw dust to glass. Multi-purpose and always useful.

Cement Mixer Core

$10,000 — Dust + Stone → Cement ($30)

Combines dust and stone for the concrete brick chain. Best stone processing path.

Clay Mixer Situational

$20,000 — 2 Dust → Clay ($50)

Only needed for the ceramic casing → superconductor path.

Ceramic Furnace Situational

$30,000 — Clay → Ceramic Casing ($150)

Fires clay into ceramic casings. Required for superconductors.

Glass Processing

Lens Cutter Situational

$75,000 — Glass → Lens (+$50)

Only needed for the optics → laser chain.

Optics Machine Situational

$300,000 — Lens + Pipe → Optics (×1.25)

Feeds the Laser Maker. Requires both lens and pipe inputs.

Sifting

Sifter Replaced by Nano

$4,000 — 10% Dust → random ore

Dust is not destroyed on miss. Becomes obsolete once you have the Nano Sifter.

Nano Sifter Prestige

⭐ 1 Medal — 16.6% sift, rarer ores

Direct upgrade to Sifter. Higher success rate and better ore distribution.

Gem Machines

Prospectors (×5) Core

$2K–$30K each — 5% per prospector, stack to 22.6%

Always run stones through all 5 prospectors. Expected gem EV = $141.03 per stone. Each checks remaining stones after the prior.

Gem Cutter Core

$20,000 — Uncut Gem → Cut Gem (×1.4)

Always polish before cutting. Polish (+$10) then cut (×1.4) maximizes value.

Prismatic Gem Crucible Situational

$100,000 — 2 Gems → Prismatic (×1.15)

Only ×1.15 multiplier. Marginal gain — use only with cheap surplus gems or for the amulet chain.

Combination Machines

Frame Maker

$10,000 — Bar + Bolts → Frame (×1.25)

Frames feed casings and amulets.

Ring Maker

$15,000 — Gem + Coil → Ring (×1.7)

Excellent multiplier. Recipe: Gem + Coil. Rings feed into amulets.

Circuit Maker

$20,000 — Glass + Coil → Circuit (×2.0)

Strong multiplier. Circuits feed tablets and lasers.

Casing Machine

$50,000 — Frame + Bolts + Plate → Casing (×1.30)

Three inputs. Casings are critical for engines, power cores, tablets, explosives, and electromagnets.

Alloy Furnace

$100,000 — 2 Bars → Alloy (×1.2)

Combines two bars. Feeds superconductors.

Magnetic Machine

$120,000 — Coil + Casing → Electromagnet (×1.5)

Solid multiplier. Electromagnets feed power cores.

Explosives

⚠️ Generally not worth it. The explosives chain requires dedicated dust production and consumes casings. The return is marginal compared to other uses of those materials.

Blasting Powder Chamber

$15,000 — Metal Dust + Stone Dust → Blasting Powder ($52)

Explosives Maker

$15,000 — Blasting Powder + Casing → Explosives (×1.5)

Blasting Powder Refiner

$2,500,000 — Blasting Powder + Metal Dust → +$15
