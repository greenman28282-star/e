🔧 Machine Logic & Obsolescence Guide

Confused about whether a new machine replaces an old one? This section clarifies every machine interaction. ✅ Core = always needed. 🔗 Useful = needed for recipes. ⚠️ Situational = use case dependent. ❌ Trap = never buy.

⚠️ Key Principle: Very few machines become obsolete! Most machines serve different recipe chains rather than replacing each other. The biggest mistake players make is thinking a new machine replaces an older one, when actually both are needed for different end-game products.

⛏️ Ore Processing (Pre-Smelting)

Ore Cleaner ✅ Core

Adds $10 to uncleaned ores. Always apply before smelting — the $10 compounds through ALL subsequent multipliers (smelt ×1.2, temper ×2, etc.).

Feeds: Polisher → Smelter | Obsolete? Never. Always needed.

Polisher ✅ Core

Adds $10 to any unpolished item. Used on ores (before smelting) AND gems (before cutting). Two completely different uses!

Feeds: Ores → Smelter, Gems → Gem Cutter | Obsolete? Never. Dual purpose.

🔥 Smelting

Ore Smelter ✅ Core

Ore → Bar (×1.2 value) + Stone byproduct. The stone is where gems and concrete bricks come from — hugely valuable.

Feeds: Bar → Tempering/Coiler/Plate/Bolts, Stone → Prospectors | Obsolete? Never.

Blast Furnace ❌ Trap

Ore → Bar (×0.9 value) + Stone. Costs $25,000 but makes bars 10% LESS valuable than the $1,500 Ore Smelter. There is NO scenario where this is better.

⛔ NEVER BUY. Ore Smelter is strictly superior in every way.

🔩 Bar Processing — Do I Need Both Coiler AND Plate Stamper?

💡 YES — both are needed! Coiler and Plate Stamper both add $20 to bars, which often confuses players. But they serve completely different recipe chains. Neither makes the other obsolete.

Coiler 🔗 Useful

Bar → Coil (+$20). Best early-game product. Later, Coils feed into Circuits (×2.0), Rings (×1.75), and Electromagnets (×1.5).

Feeds into: Circuits, Rings, Electromagnets | Obsolete? No! Needed for Tablets (via Circuit) and Power Cores (via Electromagnet).

Plate Stamper 🔗 Useful

Bar → Plate (+$20). Unlocks the entire metalwork chain: Pipes, Mech Parts, Filigree, and Casings (via Frames).

Feeds into: Pipes, Mech Parts, Filigree, Frames → Casings | Obsolete? No! Foundation of the entire main-line progression.

Bolt Machine 🔗 Useful

Bar → Bolts (+$15). Only $3K. Needed for Frames, which are needed for Casings, which are needed for EVERYTHING end-game.

Feeds into: Frame Maker → Casing Machine | Obsolete? No! Critical for all casing-based products.

Tempering Forge ✅ Core

Doubles untempered bar value (×2). This is NOT a replacement for smelting — it goes AFTER the smelter. Apply to bars before making Plates/Coils/Bolts.

Fed by: Ore Smelter (bar output) | Obsolete? Never. Massive multiplier.
✅ Does NOT replace Ore Smelter — you still smelt first, THEN temper.

🔧 Plate Products — Does Mech Parts Make Pipes Obsolete?

💡 NO! Mech Parts add more per plate ($30 vs $20), so they're better for direct selling. But Pipes are needed for Engines and Optics. Keep both.

Pipe Maker 🔗 Useful

Plate → Pipe (+$20). Lower value-add than Mech Parts, but Pipes are a required input for Engines (×1.2) and Optics (×1.25, for Lasers).

Feeds into: Engines, Optics → Lasers | Obsolete? No! Required for Engine and Laser recipes.

Mech Parts Maker 🔗 Useful

Plate → Mech Parts (+$30). Best $/plate for direct selling. Also a required input for Engines (×1.2).

Feeds into: Engines | Obsolete? No! Needed for Engine recipe alongside Pipes.

Filigree Cutter ⚠️ Situational

Plate → Filigree (+20% value). Only useful if you're making Gliders (Filigree + Amulet = ×1.5). Otherwise, Mech Parts give more per plate.

Feeds into: Gliders | Obsolete? N/A — niche use for Glider recipes only.

🧱 Stone Processing — Do Concrete Bricks Replace Fired Bricks?

💡 MOSTLY YES — Polished concrete bricks ($370/pair) are ~4× more valuable than two polished fired bricks ($92/pair). Once you have the Cement Mixer, you should always prefer concrete bricks. But you still need fired bricks for odd surviving stones that can't be paired, and the Kiln is still needed for concrete brick processing.

Crusher ✅ Core

Stone → Dust ($25 base value). Required for bricks, cement, clay, glass, rings, and blasting powder. Literally everything in stone processing starts here. Polish the dust (+$10) if selling directly.

Feeds into: Everything in stone processing | Obsolete? Never.

Brick Mold ✅ Core

Dust → Brick ($30) OR Cement → Concrete Brick ($300). Same machine, two inputs. You need it for BOTH regular and concrete bricks. Always polish bricks before kilning or selling (+$10).

Fed by: Crusher (dust) or Cement Mixer (cement) | Obsolete? Never. Does double duty.

Kiln ✅ Core

Bricks → Fired Bricks (×1.2 on base value, before polish) AND Dust → Glass ($30). Still needed after Cement Mixer because concrete bricks also pass through the Kiln for the final boost. Polish bricks after kilning for best results.

Feeds into: Glass → Circuits/Tablets, Fired/Concrete Bricks → Seller | Obsolete? Never.
✅ Cement Mixer does NOT replace the Kiln — concrete bricks still need to be kilned.

Cement Mixer ✅ Core

Dust + Stone → Cement ($50). The cement goes to Brick Mold to make concrete bricks ($300). Biggest single upgrade in the game.

Fed by: Crusher + raw stone | Feeds: Brick Mold → Kiln → Polish → Seller
✅ Makes fired-only bricks MOSTLY obsolete (~4× value per stone pair), but doesn't replace Crusher, Brick Mold, or Kiln.

Clay Mixer ⚠️ Situational

2 Dust → Clay Block ($50). Only useful as input for Ceramic Furnace → Superconductors. Don't confuse with Cement Mixer!

Feeds into: Ceramic Furnace → Superconductors | Obsolete? N/A — specific to SC recipe.

Ceramic Furnace ⚠️ Situational

Clay Block → Ceramic Casing ($150). Only useful as a component for Superconductors.

Feeds into: Superconductor Constructor | Obsolete? N/A — specific to SC recipe.

🔮 Glass Processing

Lens Cutter ⚠️ Situational

Glass → Lens (+$50). Only useful as input for Optics Machine → Lasers. $75K is steep for a niche upgrade.

Fed by: Kiln (glass) | Feeds: Optics Machine → Laser Maker

Optics Machine ⚠️ Situational

Lens + Pipe → Optics (×1.25 combined). Only useful as input for Laser Maker. Requires both Lens Cutter and Pipe Maker.

Fed by: Lens Cutter + Pipe Maker | Feeds: Laser Maker

🔄 Sifting & Recycling

Sifter ⚠️ Situational

10% chance: Dust → random low-tier ore (Tin, Iron, Cobalt, etc.). Dust is NOT destroyed on failure. Free bonus ore, but low success rate.

Fed by: Crusher (dust) | Feeds: Sifted ore → back to main line
⚠️ Becomes fully obsolete when you get the Nano Sifter (prestige). Replace immediately.

Nano Sifter ⭐ Prestige

16.6% sift rate (vs 10%) AND produces rarer ores (up to Iridium/Palladium). Strictly superior replacement for the regular Sifter.

Replaces: Sifter | Cost: 1 Medal (cheapest prestige machine)
✅ Makes regular Sifter completely obsolete. Get this FIRST after prestige.

💎 Gem Machines — Do They Replace Each Other?

⚡ Throughput Optimization

Belt speed is fixed — each conveyor segment handles one item at a time. To increase throughput:

  • Parallel lines: Use 3-Way or L/R Splitters to create 2-3 parallel processing lines that feed into the same seller. This 2-3× your output.
  • Multiple sellers: A single seller can bottleneck your entire factory. Add 2nd/3rd sellers for different product types (main product, gems, bricks).
  • Unloader belt upgrades: 2nd output belt ($100K) doubles feed rate. 3rd belt ($1M) triples it. This is often more impactful than adding machines.
  • Minimize belt length: Every conveyor segment adds transit time. Keep machines close together and use elevators for vertical shortcuts.
  • Overflow splitters at bottlenecks: Place an overflow splitter before any machine that processes slowly — the overflow path goes to a buffer or alternate machine.

💡 Pro Tips & Quick Reference

Key optimization rules. See individual sections above for detailed explanations.

🔑 The Golden Rules:
#0 — ALWAYS prospect stones before processing them ($141 gem EV per stone with all 5 prospectors)
#1 — Always Clean + Polish ores BEFORE smelting (the +$20 compounds through all subsequent multipliers)
#2 — Always QA your final products last (20% on your most expensive item = max gain)
#3 — Never sell raw stones (prospected + concrete bricked = $180+/stone plus gem EV)
💰 Diamond Prospector has the highest EV per stone.
Diamond's $1,500 base value (→ $2,114 cut) means the Diamond Prospector alone contributes ~$86 of the ~$141 total prospector EV per stone. It costs $30K but pays for itself in just ~350 stones. Buy it ASAP.
🔗 Why Plates over Coils for the main line?
Both add +$20 to bars. But plates unlock Pipes (+$20), Mech Parts (+$30), and Filigree (+20%). Coils feed into Circuits and Rings. The plate path has more branches and higher single-line value.
📊 The "Per Ore" Metric:
Always compare value per ore consumed, not total product value. Include prospector gem EV + concrete brick byproducts. But also consider throughput — a simpler product at higher volume can out-earn a complex one. See the Comparison Table.
